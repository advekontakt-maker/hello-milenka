# Simple Hello World Website

This is a minimal static website that displays "Kocham Cię Milenka".

**Live site:** https://advekontakt-maker.github.io/hello-milenka/

Files:
- `index.html` — main page
- `styles.css` — basic styles

How to view locally:

Option 1 — open directly:
- Double-click `index.html` in your file explorer to open in your default browser.

Option 2 — run a simple local HTTP server (recommended):

If you have Python installed (works on Windows):

```powershell
cd "c:/Users/48512/Desktop/Nowy folder (2)"
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

Option 3 — VS Code Live Server:
- Install the Live Server extension and click "Go Live" in the status bar.

That's it — enjoy the simple page.
